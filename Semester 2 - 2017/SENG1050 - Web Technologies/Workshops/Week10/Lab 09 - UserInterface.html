Task 3 - More XSLT : basic

  1. Given the following XML document: 
    
<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE recipe_book [
	<!ELEMENT recipe_book (recipe+)>
	<!ELEMENT recipe (title,author,year?,ingredient*)>
	<!ELEMENT title (#PCDATA)>
	<!ELEMENT author (#PCDATA)>
	<!ELEMENT year (#PCDATA)>
	<!ELEMENT ingredient (#PCDATA)>
	<!ATTLIST recipe category CDATA "dessert">
	<!ATTLIST recipe_book publisher CDATA #REQUIRED>
]>

<recipe_book publisher="Book Co.">
	<recipe category="main">
		<title>Lasagne</title>
		<author>Garfield the cat</author>
		<year>1994</year>
		<ingredient>Cheese</ingredient>
	</recipe>
	<recipe>
		<title>Pavlova</title>
		<author>Helen G.</author>
		<year>1994</year>
		<ingredient>Eggs</ingredient>
		<ingredient>Sugar</ingredient>
	</recipe>
</recipe_book>
  2. Create an XML tree to represent the structure of this document (see Lecture 3 - 02, slides 39-42). This will help you understand the XPath commands needed in your XSLT page.
  3. Create an XSLT stylesheet to tranform the page as follows:
    • Show a main heading of recipes and then two subheading matching to the two types of recipes (main and dessert).
    • For each individual recipe, show all of the information from the XML document. Try rearranging the order to help you understand how the XSLT page creates a new 'tree' from the original document.
    • Ask your demonstrator for help if you are having difficulties.

Task 4 - XSLT : Learning useful xslt

Some browsers do not support following codes.

  1. For Web links 
    	<a href='{your XPath}'><xsl:value-of select="your XPath"/></a>
  2. For image 
    	
	<img alt="some message">
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
	</img>
  3. For audio 
    						
	<audio controls="controls">
		<source type="audio/mpeg">
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
		</source>
	</audio>
  4. For video 
    						
	<video width="320" height="240" controls="controls">
		<source type="audio/mp4">
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
		</source>
	</video>
  5. For YouTube 
    						
	<embed width="560" height="315">
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
	</embed>
    Or 
    	
	<iframe width="560" height="315">
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
	</iframe>
  6. For other media 
    						
	<embed>
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
	</embed>
    Or 
    	
	<object>
		<xsl:attribute name="src">
			<xsl:value-of select="your XPath"/>			
		</xsl:attribute>			
	</object>
