1. Specifications:
  - C++ (gcc 7.3.0)
  - Cygwin version 2.11.2(0.329/5/3) x86_64

2. How to compile:
  - navigating to the directory
  - enter 'make'
  - enter the filename when prompted (and only the name of the file - the application automatically adds the directory address)
  - enter '0' (for encryption) or '1' (for decryption) when prompted
