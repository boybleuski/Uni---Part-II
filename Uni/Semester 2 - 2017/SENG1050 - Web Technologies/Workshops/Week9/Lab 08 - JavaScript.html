Task 2 - Understand and write a form validation script

In English, the test of the if is broken into two parts: if (A OR B) where A is "if no boxes are checked at all" and B is "if more than 1 box is checked". If either of these cases is true, then it is not the case that one and only one box was checked, so we issue and alert and return false. Otherwise (else), we thank the user and return true.

Task 3 - Learning new, useful JavaScript tools

  • Check that a textbox contains only numerical data. Use a form to construct a textbox (at least 10 characters) and write a JavaScript function to check that the input is numerical. In your function you can make use of the following variable 
    var pattern = /^-?\d+(\.\d+)?$/;

    The interpretation is;
    ^-? At the start of the string you may have a minus sign
    \d+ Then you will have one or more digits
    (\.\d+)? Then, optionally, a decimal point followed by one or more digits
    $ Then the string ends.
    To test, use var a = pattern.test(getElementById("id").value) and check to see if a contains "true" or "false".
  • Calculate the length of a text string. From the lecture notes, make use of document.getElementById("id").value.length. Make sure your textbox has an identifier ("id"). Use document.write() to print out the length of the text string.
  • Calculate the position of a character within a string. JavaScript's indexOf allows us to determine the position of a character (or even a substring) within a string of text. For example 
     var a = "1 1 was a racehorse, 2 2 was 1 2. 1 1 1 1 race 1 day, 2 2 1 1 2.";
    
 var b = a.indexOf("w");

    The value of the variable b is now 4 (remember JavaScript begins counting from 0).
    Choose a character from the textbox with identity "id" above and use indexOf to determine its position.

Task 4 - Implement a form validation script

Much time and bandwidth can be saved by performing simple checks on a form at the client side before the form is transmitted. The exercises in Task 2 allow for a simple form validation script.
  1. Construct a simple HTML form with input boxes for name, password, email address and telephone number.
  2. Make sure the password is at least 6 characters long. You will need to use an if statement to determine if the password in insufficient and, if so, you may like to notify the reader by means of an alert box.
  3. Make sure the box for telephone number contains only integers. (You do not need to check for negative numbers or decimals).
  4. Make sure the email address contains the "@" symbol.
  5. Consider making the script more sophisticated by allowing "+" and "( )" characters in telephone numbers to allow for international calls or by making sure that a "." appears as a character after "@" in an email address.
