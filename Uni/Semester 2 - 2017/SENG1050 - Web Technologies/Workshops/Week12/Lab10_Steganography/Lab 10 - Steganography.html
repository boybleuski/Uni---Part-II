Task 1 - Steganography

Cryptography for confidentiality encrypts a message so that even if it is intercepted, it cannot be read. Steganography hides the fact that the message exists at all. It is often used in text or software to combat pirating.

  1. Download the file steg.zip and place it in a newly created directory.
  2. Extract the files from the .zip archive
  3. Open S-Tools.exe
  4. Click and drag zebras.bmp into the S-Tools window (expand the graphic to see the photo of zebras if necessary).
  5. Right click on the image and select Reveal. When prompted, enter the code 'abc' (without the quotes) and confirm.
  6. Study the revealed files.
  7. Repeat the procedure with the file chi_in_bear.gif. Use the password 'strange' (without the quotes).
  8. Read the article Clair.pdf
